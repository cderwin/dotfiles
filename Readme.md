My dotfiles
======

## How to Install

Pretty simple.  Just `make install`.  The only dependency is the [Gnu Stow](https://www.gnu.org/software/stow/) tool for symlink management.

## Dependencies

In general, the following tools are assumed to be installed:
```bash
- zsh
- fd
- fzf
- rg
- bat
- nvim
- git
- uv
- nushell
- zoxide
- atuin
- broot
- starship
```
